# voidborne

A story based game where void enters a world ruled by light.

**currently a heavy work in progress**

**will be available in ~a month on itch**

stay tuned!!

AD/Left Right – Move

W/Up Arrow – Look up (for attack)

S/Down Arrow – Look down (for pogo)

Ctrl – Dash

Space – Jump

E/Right Click – Heal

Q - Interact

X/Left Click – Attack
